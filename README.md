# 📦 Inventory Management System

A full-stack inventory management application built with React and Node.js.

## 📂 Project Structure
## 🚀 Quick Start

### Backend Setup

```bash
cd backend
npm install
npm run dev
```

Backend runs on `http://localhost:5000`

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on `http://localhost:5173`

## ✨ Features

### Backend API

- ✅ REST API with CRUD operations
- ✅ Product management endpoints
- ✅ Data validation
- ✅ Inventory statistics
- ✅ Stock level tracking

### Frontend Dashboard

- ✅ Create products (form with validation)
- ✅ Read products (table display)
- ✅ Update products (inline editing)
- ✅ Delete products (with confirmation)
- ✅ Search by name/SKU
- ✅ Filter by category
- ✅ Sort by price/quantity
- ✅ Stock level indicators
- ✅ Inventory value calculation
- ✅ Real-time statistics

## 🛠️ Tech Stack

**Backend:**
- Node.js
- Express.js
- CORS
- dotenv

**Frontend:**
- React 18
- Vite
- Tailwind CSS
- Lucide Icons
